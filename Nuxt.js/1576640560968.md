## CSS 配置

**Reference**

https://zh.nuxtjs.org/api/configuration-css


**Install**

```
npm install --save-dev node-sass sass-loader
```

**Config**

> nuxt.config.js

```
module.exports = {
  css: [
    // 直接加载一个 Node.js 模块。（在这里它是一个 Sass 文件）
    'bulma',
    // 项目里要用的 CSS 文件
    '@/assets/css/main.css',
    // 项目里要使用的 SCSS 文件
    '@/assets/css/main.scss'
  ]
}
```
